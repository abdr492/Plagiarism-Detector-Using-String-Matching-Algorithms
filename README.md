# 🧠 Plagiarism Detector Using String Matching Algorithms

A complete **DSA-based plagiarism detection system** that detects:

- ✅ Exact copied content
- ✅ Near-duplicate text
- ✅ Lightly paraphrased content
- ✅ Similar phrases and sentence structures

This project demonstrates powerful **String Matching Algorithms**, **Hashing**, **Sliding Window**, **Fingerprinting**, and **Similarity Detection Techniques** used in real-world plagiarism detection systems.

---

# 🚀 Project Objective

The main objective of this project is to build an efficient plagiarism detection pipeline using:

- KMP Algorithm
- Rabin-Karp Algorithm
- Hashing
- Shingling
- Winnowing Fingerprinting
- MinHash & LSH
- TF-IDF Similarity
- Jaccard Similarity

The system compares documents and generates:

- 📊 Plagiarism Percentage
- 📝 Similarity Reports
- 🔍 Matched Content
- 📑 Highlighted Evidence

Educational institutions, publishers, EdTech platforms, and content review systems need automated tools to identify copied or duplicated content.

# 🔍 Algorithms Used

## 🔹 KMP Algorithm

Efficient pattern matching algorithm using:

* LPS Array
* Prefix-Suffix Optimization

Time Complexity:

```text
O(N + M)
```

---

## 🔹 Rabin-Karp Algorithm

Uses:

* Rolling Hash
* Sliding Window

Time Complexity:

```text
Average: O(N + M)
Worst: O(NM)
```

---

## 🔹 Winnowing Fingerprinting

Generates fingerprints for:

* Near duplicate detection
* Robust matching

Used heavily in modern plagiarism systems.

---

## 🔹 MinHash + LSH

Used for:

* Large-scale document comparison
* Faster candidate retrieval

---

## 🔹 TF-IDF + Cosine Similarity

Detects semantic similarity between documents.
